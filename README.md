# 🎬 Movie Recommendation System

## 📌 Project Overview
This project was developed as part of the Artificial Intelligence Internship at Codec Technologies. The system recommends movies based on genre similarity using Machine Learning techniques. It analyzes movie genres and suggests the most relevant movies according to user preferences.

## 🚀 Features
- Personalized Movie Recommendations
- Genre-Based Similarity Matching
- TF-IDF Vectorization
- Cosine Similarity Algorithm
- Fast and Accurate Recommendations

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## 📂 Project Structure
- movie_recommendation.py
- README.md

## 🎯 How It Works
1. Load movie titles and genres.
2. Convert genre information into numerical features using TF-IDF.
3. Calculate similarity between movies using Cosine Similarity.
4. Accept a movie name from the user.
5. Display the top recommended similar movies.
